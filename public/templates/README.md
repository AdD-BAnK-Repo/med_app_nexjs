# คำแนะนำในการตั้งค่า Template เอกสาร Word

1. สร้างไฟล์ Word (`.docx`) ที่จะใช้เป็นแบบฟอร์ม (Template) สำหรับใบขายเวร/แลกเวร
2. บันทึกไฟล์ชื่อ `template_shift.docx` แล้วนำมาวางไว้ที่โฟลเดอร์นี้ (`public/templates/template_shift.docx`)
3. ในจุดที่ต้องการให้ระบบนำข้อมูลมาเติมอัตโนมัติ ให้ใส่ Tag ตามรูปแบบนี้ในไฟล์ Word:
   - `{requester}` - ชื่อผู้ขอขายเวร
   - `{substitute}` - ชื่อผู้รับเวรแทน
   - `{day}` - วันที่ (เฉพาะตัวเลข)
   - `{month}` - เดือน (แบบเต็ม)
   - `{year}` - ปี พ.ศ.
   - `{shiftTime}` - เวลาเวร (เช้า/บ่าย/ดึก)
   - `{orderNumber}` - เลขที่คำสั่ง
   - `{orderDate}` - วันที่ของคำสั่ง

ตัวอย่างข้อความในไฟล์ Word:
> ข้าพเจ้า {requester} มีความประสงค์ขอเปลี่ยนเวรปฏิบัติงาน ในวันที่ {day} เดือน {month} พ.ศ. {year} เวร {shiftTime} โดยมี {substitute} ปฏิบัติงานแทน
> อ้างอิงคำสั่งเลขที่ {orderNumber} ลงวันที่ {orderDate}

หลังจากนำไฟล์ของจริงมาใส่แล้ว ระบบจะสามารถออกเอกสาร Word (.docx) ผ่านหน้าเว็บได้ทันที
